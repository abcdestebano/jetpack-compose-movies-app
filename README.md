# Jetpack Compose Movies App

Project create with Jetpack Compose

Data taken from https://www.themoviedb.org creating an developer account and requesting an api-key

## Libraries Used :
- Jetpack Compose
- ViewModel
- Coroutines
- Retrofit2
- Coil
